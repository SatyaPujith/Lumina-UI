# Lumina UI

Lumina is a sleek, dynamic, and premium web application for discovering and hosting local experiences and events. Built with modern web technologies and a focus on visual excellence, Lumina provides users with a seamless way to browse categorized events, connect with hosts, and schedule experiences.

## Getting Started

Follow these steps to get a copy of Lumina running on your local machine for development and testing purposes.

### Prerequisites

Ensure you have Node.js and an appropriate package manager (npm) installed.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SatyaPujith/Lumina-UI.git
   ```

2. **Navigate to the core directory:**
   ```bash
   cd Lumina-UI
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **View the Application:**
   Open your browser and navigate to `http://localhost:5173/` (or the local port provided in your terminal).
